Introduction

The Nintendo Switch is a hybrid gaming console that can be played both at home and on the go. Here are some fun facts about the console, and why people would want to play it in the first place:

  • It is known for its ability to be docked to a TV, and played in handheld mode.
  • This console features a new 7-inch OLED screen, and is perfect for playing with friends and family.
  • It houses some of gaming's most beloved titles in its library, including Mario Kart 8 Deluxe, Super Smash Bros Ultimate, Zelda: Breath of the Wild, Super Mario Odyssey, and Animal Crossing New Horizons.

Setting Up

To set up your Nintendo Switch for the first time, follow these steps:

  • Connect the USB plug from the Nintendo Switch AC adapter into the USB connector on the bottom of the console, then connect the AC adapter into the wall outlet.
  • Attach the Joy-Con controllers to the Nintendo Switch console.
  • Follow the on-screen prompts to complete the first-time setup. Some of these settings can be configured later.
  • System language and region settings
  • Internet Settings (optional)
  • Time Zone
  • Set your icon and nickname or Transfer Users and Save Data (this option is available for consoles with system menu version 4.0.0. or higher)
  • Parental Controls (optional)

Charging

  • Take the charger out of the box and connect to a power outlet, then connect your console to the USB-C.
  • Nintendo Switch systems take approximately 3 hours to fully charge when they are powered off or in sleep mode. Charging time will be longer if the system is in use.
  • The battery duration varies depending on system model and the on the software application and functions being used.
  • For Nintendo Switch, model number: HAC-001 [product serial number begins with "XAW"], the battery life is approximately 2.5 to 6.5 hours.
  • For Nintendo Switch, model number: HAC-001(-01) [product serial number begins with "XKW"], the battery life is approximately 4.5 to 9 hours.
  • For Nintendo Switch Lite, model number: HDH-001 [product serial number begins with "XJW"], the battery life is approximately 3 to 7 hours.

Playing Software

  • There are two types of games that you can purchase on the Nintendo Switch: Physical Copies and Digital Copies.
  • You can purchase physical games in in-person stores, such as GameStop, any local video-game stores, or at online marketplaces such as Amazon or Ebay (for cheaper second hand games).
  • Meanwhile, you can purchase Digital Games on the Nintendo eShop, which will be discussed in more detail later.
  • To play physical games, simply pop the cartridge into the cartridge holder on the top right of the console.
  • To play digital games, ensure you have enough storage on the console to download the game, and you can play by clicking on the game icon :D!

Connecting Online

Here are the steps to connect a Nintendo Switch console to Wi-Fi in order to use online services.

  • Before you start, you will need:
  • Your Nintendo Switch console
  • The wireless network name (SSID)
  • The wireless network password, if it requires one
    •
  • Place the Nintendo Switch within 10 feet of the wireless router so there will be a strong wireless signal.
  • Select System Settings from the HOME Menu on the Nintendo Switch console.
  • Select Internet, and then Internet Settings. It will automatically search for nearby Wi-Fi signals.
  • Select the appropriate network name (SSID) from the list.
  • Use the onscreen keyboard to enter the network's wireless password, if one is required.
  • The Nintendo Switch will test the Internet connection. Once the test is complete, select OK to complete the process.

Account Settings

Here are some things you should know about your Nintendo Account and game access.

  • A Nintendo Account is an account that provides access to certain features on Nintendo Switch, including access to the Nintendo eShop, the Nintendo Switch Online membership service, and the My Nintendo rewards program.
  • Up to 8 Nintendo Accounts can be linked to a Nintendo Switch console at a time.
  • You can link your Nintendo Account to multiple consoles. You can play your virtual game cards on up to two consoles, and anyone else that uses those consoles can also play them.
  • Physical games can be played on any console you like. Save data for physical and digital titles can be transferred between consoles using Save Cloud Data backup (available to Nintendo Switch Online members) or the on-device transfer process.

Friends

This is how you add friends to your Nintendo Switch!

  • Select your user icon (My Page) on the HOME Menu, and then select Add Friend.
  • From this screen, you can accept pending friend requests that have been sent to you, view pending friend requests you have sent, or use the following methods to send friend requests.
  • Friend Suggestions: Select this option to send friend requests to users that you are friends with on Nintendo smart-device apps (Super Mario Run, etc.), Wii U, or Nintendo 3DS.
  • Search for Local Users: Select this option to exchange friend requests with near-by users.
  • Search for Users You Played With: Select this option to send friend requests to users you have played with in internet matches or other online play.
  • Search with Friend Code: Select this option to send a friend request to another user by entering their friend code.

Parental Controls

If you are planning to allow your kid to play on this console, you may be wondering if there are any parental controls available and how to set them up.

  • Depending on the type of software, purchases can be restricted through the Nintendo Account and/or Nintendo Network ID (NNID) that is linked to the child's account.
  • Parental controls for off-device and Nintendo Switch eShop purchases (for both child and supervised accounts) can be set up through the parent/guardian's Nintendo Account.
  • The smart device app, which can be downloaded on the app store, provides access to the most restriction options and features.
  • Set limits on play time, view gameplay history, and whitelist individual games to ignore restrictions.
  • Set age rating restrictions, limit communication between players, and set social media restrictions.
    •
  • System settings can be used without a smart device.
  • Set age rating restrictions, limit communication between players, and set social media restrictions.
    •
  • Nintendo Account settings can be used for separate restrictions.
  • Restrictions on viewing and purchasing content in the eShop, friend suggestions, and sharing information with third-parties are managed through Nintendo Account settings.

Using the eShop

Here are some steps to buy games or Downloadable Content (DLC / digital add-on content for a game) from Nintendo eShop by using existing funds, adding funds with a Nintendo eShop card, or redeeming a Nintendo Switch Game Voucher. The full game is required to access DLC. Before you start, it is important to note that both an Internet connection and Nintendo Account are required to access the Nintendo Switch eShop.

  • Locate the game you want to purchase, and select it to reach the game details page:
  • Select Nintendo eShop on the HOME Menu to launch the Nintendo eShop.
  • If there are multiple user accounts on your system, select the account you want to use.
  • Select the magnifying glass in the upper-left corner to access the Search menu.
  • Enter a keyword into the keyword field and select Search to search by the keyword. A list of results will appear.
  • After selecting a game title, scroll through the game details to view any available downloadable content.
    •
  • Any DLC or In-Game Items associated with the game will also be available through the game details page. Select the individual DLC/item from the game details to proceed with purchasing it.
  • Select Proceed to Purchase. One of several different options will appear, depending on the game and your funds balance.

Data Management

You may also wonder how to transfer user and save data on Nintendo Switch, so here are a few steps! :)

  • Prepare the transfer on the source console.
  • From the HOME Menu, select System Settings, then Users, and then Transfer Your User Data.
  • Select Next, and then Next again, then select Source Console to indicate that the content will be transferred from this system.
  • Select Continue, and then use the target console for the next steps.
    •
  • Prepare to receive the data on the target console.
  • From the HOME Menu, select System Settings, then Users, and then Transfer Your User Data.
  • Select Next, then Next again, and then select Target Console to indicate that the content will be transferred to this system.
  • Select Sign in and choose to sign in using your Nintendo Account e-mail address or sign-in ID.
  • Enter either the e-mail address or sign-in ID associated with your Nintendo Account and the password, then select Sign in, and then Next.
    •
  • Complete the transfer on the source console.
  • Wait for the target system to be found, and then select Transfer.
  • The transfer process will begin, wait for this process to complete.
  • Once complete, select End to complete the process.

If you have multiple consoles or want to share with friends, you may wonder how to transfer Digital Games / Nintendo Accounts between Nintendo Switch consoles. Read the text below to find out!

  • From within the System Settings of the original console, use the option to Transfer Your User and Save Data for each account you wish to transfer.
  • You can also select Transfer Your Save Data if you want to move an individual game save file to a different system.
  • Using this option will allow you to retain game save data, including save data for games that do not support Save Data Cloud backup.
  • Once this process is complete, the associated Nintendo Account, user account, and save data is removed from the original console after being transferred to the second console.
